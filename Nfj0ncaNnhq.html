緣由

g0v summit 2016 有許多外國朋友參加，因此每間會議室都配置了兩位即時口譯老師。口譯老師的聲音要如何播送給聽眾呢？目前有專業廠商會提供專門的耳機和發送器，讓會場內聽眾收聽口譯內容，同時將音源訊號接給直播組，讓遠端聽眾也可以享受口譯服務。但由於現場的耳機只有 400 組，無法提供給所有會眾，因此需要另外架設 icecast 網路電台來補足，讓會眾只要有智慧型手機或筆電，即可透過網路收聽 icecast 口譯頻道。

為了自幹 icecast 口譯頻道，線路組已率先提供了 icecast server 機器，還剩下 icecast client 的硬體設備沒有著落，所以，接下來就是沒有人的事了！

跳坑區

  • 筆電 + 電源線 x 3
    •

    用途：擔任 icecast client

    規格：mac 或 win 都可，但要有功能正常可使用的 ethernet 有線網路孔

    認領（請附上廠牌型號）：

  • 48k 以上 USB 麥克風 + 腳架 x 3
    •

    用途：收口譯老師的語音，播送到 icecast 頻道給遠端參與的聽眾

    規格：需要 48k 以上錄音等級，不然音質會很差。需要是 usb 接頭，因為若是 mac 筆電 3.5mm 的孔已經被耳機佔走

    認領（請附上廠牌型號）：ET x 1 (Apogee MiC 96k) , 

  • 3.5mm 頭戴式耳機 x 6
    •

    用途：給即時口譯老師監聽演講內容，每間會議室同時有兩位口譯老師，R0, R1, R2 三間會議室會有六位同時使用

    規格：要是頭戴的，不能是勾耳的。開放式或封閉式都可以。要是有線的，不能是藍芽的。

    認領（請附上廠牌型號）：ET x 2 (i-rocks 穿脫都由你 IH05 電競耳麥) , 

  • 3.5mm 音源分接線 x 3
    •

    用途：擴充筆電的 3.5 mm 音源孔，讓兩組耳機可以同時收到音源訊號

    規格：長度不限，只要可以分接都可
