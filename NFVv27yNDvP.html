第一步：釐清「開放資料」到底是什麼

open definition 不能加任何限制，最多只能要求 attribution

關於閒置空間資料

  • 土地：內政部地政司，所有政府所有土地 data.gov polygon 都有
  • 建物：比較麻煩，之前資料最完整的其實是中華電信

地方政府開放資料分析

  • 地方政府最受矚目的資料：犯罪、交通事故等公共安全相關
  • 違法公告事業單位：衛生、勞檢、環保等
    • 要提供統編等可 reference 的 id
    • 要同時考慮追蹤訴願的現況

大量即時資料的處理方式：北市資訊局與微軟有協議，交通資料由微軟 host
