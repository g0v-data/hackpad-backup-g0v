Introduction

建立類似 diy.org 成就/任務/徽章系統的概念, 以循序漸進 + 由簡入繁 + 指令型態的項目，讓新鮮人漸漸了解 g0v 的目的, 並且讓更多的人進入到"主動" "參與" 的情境。

預計是 g0vis 視覺化呈現的一部分：g0vis 希望提供很容易讓人理解的 g0v 訊息, 而 levelup 則是試著引導人參與其口，但系統可以用 API 的形式獨立出來，讓各專案都可以利用。
