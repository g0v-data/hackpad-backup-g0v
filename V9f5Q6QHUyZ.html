藥要看 API (alpha)

基本上每個前端的網址前面加上 /api 就是對應的 api 路徑 ( 像是 http://drugs.olc.tw/drugs/index/page:2 的 api 位置就是 http://drugs.olc.tw/api/drugs/index/page:2 )，網頁最下方也可以看到 "本頁 API" 的連結，任何的 api 網址加上 ?pretty 就會用比較容易閱讀的格式呈現
