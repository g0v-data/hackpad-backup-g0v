API List

目前已經藉由clkao的townlist資料，完成部分API：

  1. Get all data: https://evacuation.herokuapp.com/sanctuaryList [GET Method]
  2. Get county list: https://evacuation.herokuapp.com/counties [GET Method]
  3. Get town list by county id: https://evacuation.herokuapp.com/countyID/towns [GET Method]
  4. Get village list by town id: https://evacuation.herokuapp.com/townID/villages [GET Method]
  5. Get sanctuary list by town id and village id: https://evacuation.herokuapp.com/townID/villageID/sanctuaries [GET Method]
  6. Update sanctuary list by town id and village id: https://evacuation.herokuapp.com/townID/villageID/sanctuaries [PUT Method]

WHY?

    昨天(summit 2016第一天)才知道KNY介紹的強震即時警報APP。同時，KNY也提到了消防署其實早就有製作好各縣市各鄉里的疏散避難地圖的疏散指南。 如果能夠將疏散避難地圖做成一個傳送地理資訊並回傳即時的疏散方向及路線，就可以結合其它的警報APP，盡可能的降低傷亡。
