提出由 2012 年 10 月至今，於以下列舉的平台或未列舉的其他公開專案管理平台，如 Github 、hackpad 等協作平台，實際參與專案開發、推廣或其他有助於 g0v 專案的貢獻，只需您提出有效的證明（專案網址、commit log、文字、圖片、影像、演講投影片或其他方式），我們都非常歡迎您提出申請。

英文版（加註底線的急需修改）

Do you want to come to g0v Summit 2014, but worry about not getting TICKETS before it is sold out?

We do have a community contributors program for people who make contribution to open source communities to attend g0v Summit. We will send Registration Codes to qualified applicants. Activating a Registration Code by the registration deadline (to be published later) will get an invitation to g0v Summit 2014.

Open for Application

September 15, 2014 - September 20, 2014 (announcement on September 22)

Qualifications

Your participation in an open source project since October 2012 makes you eligible for applying for a Registration Code. The open source project and your participation must meet the following conditions:

The project source code must be released with a license approved by Open Source Initiative [1].

    The project must be hosted on a public repository like Github (github.com), Google Code (code.google.com), SourceForge (sourceforge.net), or others.

    You must provide proofs of your contributions in forms of web pages, commit logs, mail contents, photos, or others that are publicly accessible. Your contributions can be codes, documentation, promotion campaign, or any other activities that help the project in a positive way.

Qualified applicants will be granted a Registration Code on a first come first serve basis until Registration Codes run out. We will release 150 codes for community contributors program.
