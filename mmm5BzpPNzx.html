一日三餐

主旨

利用動民主投票元件，解決小情侶小夫妻三五好友大食團……今天到底要吃什麼的問題

使用方式

  1. 登入
  2. 建立食物清單，清單來源
    • 手動輸入於 list RGB
    • favorites in google maps restaurant
  1. 整理食物清單，可能的 tag
    • 類型：中式 / 西式 / 大餐 / 小吃 ... etc.
    • 價位：每人平均 150↓ / 300↓ / 500↓ / 1000↓ / 3000↓
    • 距離：走路可到 / 機車可到 / 捷運可到 / 開車可到 / 外送
    • 口感：清爽 / 油膩 / 辛辣 / ...
    • 菜單內容：全素 / 奶蛋素 / 無脊椎素 / 無豚 / 無牛 / 無羊 / 無蹼 / 無雞鴨 / ...
    • 食材類型：在地食材 / 安全食材 / 有機食材 / 放牧法 / OO耕法 / 契作食材 / 公平貿易食材 / ...
    • 其他自訂 tag
  1. 共用清單
    • public
    • specific user
  1. 吃飯前投票
    1. 到之前定義好的某個清單
    2. 排序投票
    3. 送出後看大家的投票結果
    4. 依照投票結果的優先順序訂餐 / 叫外賣... etc.
