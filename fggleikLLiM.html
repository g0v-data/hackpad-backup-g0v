User Story 

列出各族群的目標、行為、相對應的 to-do 與 not-to-do

高度數位化，關心公眾事務的網民

  • 可能擁有一個以上社群網站帳號
  • 可能擁有一個以上行動裝置
  • 可能有資訊焦慮及網路成隱症
    • 例如爬個山也要帶個網路分享器
  • 不那麼熟悉政治及法律制度
  • 善於學習及使用數位科技工具解決問題
  • 認親都用ID而不是本名
  • 對於公眾議題較被動接收或接近漠視，主要使用本系統追蹤，並且用來分析各方論點及立場，以作為自己對議題，人，組織看法的依據

目標: 

  • 使用 API 製作客製化工具
    • 在網站上找到API使用手冊
  • 覺得本系統欠缺調較，想要參與本系統開發
    • 在網站上找到如何參與開發的資訊 (接到hackfold)
  • 跟中度數位化網民一樣使用本系統提供的功能

中度數位化，關心公眾事務的網民

  • 可能擁有一個以上社群網站帳號
  • 可能擁有一個以上行動裝置
  • 不那麼熟悉政治及法律制度
  • 對於公眾議題較被動接收或接近漠視，主要使用本系統追蹤，並且用來分析各方論點及立場，以作為自己對議題，人，組織看法的依據

目標:

  • 紀錄當前瀏覽的網站及對其內容的想法
    • 點擊瀏覽器的某個按鈕，輸入標籤，或留言，送出後，他可以在系統的回顧功能中找到它
  • 腦海中閃過得念頭，想暫時記錄下來
    • 點擊瀏覽器的某個按鈕，留言，送出後，他可以在系統的回顧功能中找到它
  • 看看自己對某議題的看法

社運人士

  • 對於公眾議題較主動參與，通常是行動發起人
  • 較熟系政治及法律制度
  • 可能會透過本系統的資料整理出說服文宣，尋求更多人支持他的立場
  • 可能會透過本系統進行串連，嘗試改變現狀，像是施壓代議士

記者

  • 利用本系統蒐集報導素材或

Systematic

adjective: systematic
