預定功能

這玩意大概是這六種東西的複合體

  • 書籤 : 蒐集
  • 日記 : 紀錄對特定議題當下的想法, 因始可以回顧自己看法的轉變
  • 通訊錄 : 找到正確的人
    •  政府公開通訊錄
  • 糾團 : 找戰友 (delegate to other web site.)
  • 任務追蹤 : 提案或表達意見後看事情有沒有改善
  • 成就系統 : 增加鄉民參與公眾事務的動力
  • 報表: 看視覺化數據
  • API

使用方式

  • 蒐集事件, 評論
    • 社交網路討論串 (fb, twitter, g+, plurk, youtube, etc)
    • 新聞報導
    • 可利用 LFTT 做自動分類 Use https://ifttt.com/wtf
    • 自動關聯自己社交網路上有tag的討論串
  • 分類
    • 加標籤, 標籤是種名稱，名稱只是物件的指謂
    • 寫意見
      • 針對標籤(可能是對像)
      • 針對對象
      • 針對事件
  • 定期回顧
    • 回顧自己收集的事件
      • by 人. e.x 014編年史
      • by tag. e.x 腦殘
    • 回顧相關事件(別人蒐集的事件)
      • 針對事件/標籤寫意見
  • 向目標表達意見
    • 目標為組織或個人
    • 請願/開幹 (delegate to other web site)
      • 加入別人或自己來
      • 書面表達 (傳真 or Email, 對方網站)
      • templat system, and guiding doc. (怎麼寫交戰守則)
      • 行動表達
    • 倡議
      • delegate to other web site
