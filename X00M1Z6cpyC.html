 從透明走向參與，

  • 為何我們需要「國會大代誌」？
    • 曾經我們以為人人有票投就是民主，但如果投了票，缺乏監督機制，民意代表不再代表民意，xxxxx
    • 「國會大代誌」的透明資訊、清晰介面、即時分享，提高公開度及公民參與度，運用資訊科技
  • 國會大代誌解決了什麼問題？
    • 「國會大代誌」提供立法從提案、審查、一二三讀之間所有的文件、影音資料的整合介面，降低追蹤議題的門檻，並且更容易分享。
    • 功能比較表：
      項目 ly.gov ly.g0v
      公報 內容 pdf, doc Web 瀏覽器線上閱讀
      永久連結 無 單一永久連結方便分享
      社群分享 無 單一永久連結方便分享
      議案 內容 pdf, doc Web 瀏覽器線上閱讀
      法案版本比較 無 彩色比較介面
      永久連結 無 單一永久連結方便分享
      IVOD 瀏覽器 IE only 跨瀏覽器、跨平台
      過去影音 只有立委可調閱 youtub線上備份
      手機觀看 無 iOS, Android
      互動評論/評分 無 即時評論、互動遊戲評分
      投票結果 開發中
      議案進度表 無 圖像呈現
  • 國會大代誌是如何煉成的？
  • 零時政府立院小組綜合二十個立法院網站資料，將提案、委員會審查、院會各階段文件、影音整理成結構性資料，自動化更新所有資料。
  • 國會大代誌的下一步？
    • 在將資料公開透明化的初步目標後，可提供資料探勘（Data Mining）的基礎，國會大代誌將進一步交叉比對
