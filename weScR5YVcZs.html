步驟 - 將現有作品變成 g0v 專案

程式類

  1. 選擇任一種 open source 授權，寫入 readme.md
  2. 在 repo 加入 g0v.json 檔案, 格式 https://github.com/g0v/hack.g0v.tw/blob/master/g0v.json
    1. 詞彙說明 g0v.json spec
  3.  (optional) 可以把 repo transfer 給 https://github.com/g0v
  4. 在 http://hack.g0v.tw/project 搜尋框輸入專案名稱
    • 隨便打一段字，找不到結果時會出現新增專案按鈕，按下後可輸入 github repo url
  • 這要不要和3互換呢？因為現在  hack.g0v.tw/project 無法修改專案資訊，先加入專案之後再轉移給g0v的話，會造成資訊不一致
  • ++! 像現在這樣可以嗎？
  • 是的 :D
  • :D :D 
  1. 把自己參與的作品專案加入 g0v 專案收集地 – hackfoldr 

文創類

  1. 選擇任一種 creative commons 授權
  2. 把檔案以及授權文件上傳至 g0v google drive 底下的 文化部資料夾 內適當的分類中
