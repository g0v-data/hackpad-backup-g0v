#DigitalToolbox

#{數位工具箱}

專案介紹 | Project Readme / Meta

  • 平常我們都會看到一些好用的網頁工具，很便利的改善我們工作的效率
  • 所謂"工欲善其事，必先利其器"
  • 因此想要做出一個整理這些好武器的"兵器庫"
  • 讓大家可以自由的挑選自己和手的武器~

專案名稱的部分很希望大家不令色KUSO一下，幫我取一個好玩的名子!

來龍去脈（想解決什麼問題）

靈感來源

  • 常常在逛網頁或是RSS時
  • 看到有BLOG介紹一些新奇的好工具
  • 但是看看之後就忘了
  • 等到下一次要找的時候，也不記得"關鍵字"...
  • 不如就把他們都存在一個資料庫，做好分類
  • 讓大家都可以自由查詢使用!

主旨

  • 讓新奇好用的網頁工具能夠更容易的讓大家去尋找跟使用~

牽涉領域(可補充)

  • 網頁/設計/資料庫/APP/前端/後端/

現有類似專案

  • Google webstore, 提供了小軟體或是網頁的入口讓使用者方便的連結
  • 當中也有評價跟介紹等等，但是他不是為台灣的使用者專門設計
  • 我們需要的是這些工具的"連結"、"比較"、"介紹"
  • 我們更需要統合群眾智慧的結晶
  • 讓這些淬鍊的成果，能有更好的應用
  • 所以最不希望的就是像google一樣
  • 什麼工具全都丟到資料庫上再去做分類
  • 我們要反其道而行，先找出好的工具再去分類
  • 但是實際讓要怎麼做到，就有勞我們大家一起努力!
