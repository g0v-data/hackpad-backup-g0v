#{數位工具箱}

專案介紹 | Project Readme / Meta

  • 平常我們都會看到一些好用的網頁工具，很便利的改善我們工作的效率
  • 所謂"工欲善其事，必先利其器"
  • 因此想要做出一個整理這些好武器的"兵器庫"
  • 讓大家可以自由的挑選自己和手的武器~

專案名稱的部分很希望大家不令色KUSO一下，幫我取一個好玩的名子!

來龍去脈（想解決什麼問題）

靈感來源

  • 常常在逛網頁或是RSS會看到有BLOG介紹這些
  • 新奇的好工具，但是看看之後就忘了
  • 等到下一次要找的時候，也不記得"關鍵字"...
  • 不如就把他們都存在一個資料庫，做好分類
  • 讓大家都可以自由查詢使用!

主旨

  • 讓各種網頁工具能夠更容易的讓大家去尋找跟使用~
