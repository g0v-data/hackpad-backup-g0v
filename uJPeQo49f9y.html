concept

  • 官網
    • 整個官網就等於是媒體區
    • 所有給「沒參與但想瞭解」的人看的內容都可以在這裡找到
  • hackspace
    • 整個 hack.g0v.tw 等於是黑客工作區
    • 所有給「想填坑」或「正在填坑」所需的資源都可以在這裡找到
    • 想填坑或正在填坑的人又可細分為四種
      • 技術部 - 會寫程式的
      • 文化部 - 會畫圖、設計、寫文案的
      • 研究院 - 擁有特定 domain knowledge 的
      • 工人院 - 沒特殊專長，但會上網會打字的
      • 登入狀態下，會盡量呈現符合個人背景的資訊，比方說不要塞一堆 github issue 在文化部黑客的畫面中
  • 成果和專案不同
    • 成果的敘述文字會以行銷導向、有黏力為原則
    • 專案的敘述文字會以填坑導向、快速交代工作內容為原則
    • 一個成果可能是很多專案組成，如萌典 web 專案、萌典 ios 專案，最後成果都在萌典上
    • 一個專案可能產出很多成果，如萌典俄羅斯專案包括注音遊戲、組字遊戲、詞彙遊戲
    • 成果放官網，專案放工作區
  • 官網跟 hackspace 之間以 navbar 上的一個位置相近的按鈕做 switch
  • 官網首頁是一頁式 landing page
