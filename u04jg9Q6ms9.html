#{國會大代誌 APP What has my congress done }

專案介紹 | Project Readme / Meta

  • 建立國會大代誌Android App, 使移動設備能更快速的獲得國會大代誌的資訊 

來龍去脈（想解決什麼問題）

說明

  • 鑑於對於服貿的問題, 如何讓民眾能更快速的獲得政府政策的資訊變成一個需要解決的問題。慶幸的是, 移動產品的普及使得民眾有更加多元化的方式獲取資訊。國會大代誌App的目的是除了讓民眾能更快速的檢閱資訊之外, 還能透過例如推送技術來達到全民參與政治，全民關心政治的目的。

現有類似專案

  • 其實各類新聞資源App都可以是國會大代誌APP的借鑑。國會大代誌APP所不同的地方在於它使用國會大代誌的資料庫並且允許用戶透過設定鬧鈴等功能來參與政治及其追蹤。
