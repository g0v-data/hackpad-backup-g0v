刊物宗旨

"培養、啟蒙、激發"公民意識，改變大家對於國家政策的冷漠，進而採取行動去解決目前失靈的代議制度。簡化政治論述成一般民眾能感受理解的文字。讓不同閱讀習慣的民眾，也能接收不同的聲音，改變資訊壟斷的情形。(參考企劃組)

初期以服貿為主軸，搭配其他當期重要社會議題。

第一次開會需討論的問題整理

 依照之前的共筆、討論等內容，加上設計組以上的討論，提出以下幾點問題是需要被討論的

 

1.  請企畫組決定好關於報紙的相關企劃（目標族群.發放地點.報紙的目的...等等）例 : 

  • 目標族群 : 
    • 大概抓個年齡層才能調整文案字眼難易度與報章風格設計細膩度。抓個年齡層文案組才能運用敏感字眼抓住族群的眼睛，我們不能做讓人一看就忘的刊物，我們必須引導讀者陷入思考。
  • 報導風格：
    • 你要報導什麼？你要訴說什麼？你要表現什麼？中立？偏激？詼諧？
    • 給個訴求並以這訴求為目標，訂定專欄讓文案組不會走偏，讓設計組擬訂公版，才能有效率的出刊。
  • 策略規劃：
    • 根據設定目標客群（ex如果是：資訊接收管道單一、習慣對權威全盤接收的服貿/國民黨死忠支持者；這些人坐落在哪些年齡層和職業？應有相當多研究可查詢，推論他們的身份平時會去哪些地方，在意什麼事情，能接受什麼樣的風格語氣）、市場現況分析（ex已經有哪些類似的東西了，我們的不同點/優勢/劣勢在哪裡，要和誰（如王周刊/中天）以及如何競爭）、可取得資源（ex長期人力與資金來源與預估）等以上資料規劃設計出策略，如此刊物才會有特殊性、針對性和可行性，進而產生效力。

2.  文編請訂出每期的文字量and文宣報的各個專欄(包含文宣報名稱中英文、抬頭等等喔! )


  • （如果是要穩定每期出產的刊物這怎麼能不固定呢？），這樣每邊才能綜合以上設計出公版。

  • 強調!!目前看到的企劃組欲放置的一期內容量"太大"!!(所以在上頭有放置字體大小給各位參考，在字數、內容尚須斟酌，不然即使我們照著大家需求所設計出來的公版，結果因字數等導致密密麻麻，效果下降、NO GOOD~~)

  • 依照企劃組的共筆，例 : 第四面 內容實在太多，要塞一個反諷漫畫，另外還有兩大主題，漫畫會過小，下面的內容主題也會無法完整表現。

3.  公版改變耗時、耗力，順序上不應顛倒  ==>  企劃生骨，文編長肉，美編覆皮 

  • 再次強調，沒有順序反過來這回事。這樣大家都在上班如此忙碌，現在生產出來的東西有何意義實在不明白。 這是最該堅持也不能妥協的事情，如此的方法這份刊物才能得到最大的效益也才能作出最好的設計不枉費各組如此辛苦的討論。

  • 而建議企劃組製作所謂的企劃單，設計組原有提供範本，可參考以下連結：
  • http://blog.xuite.net/benny0822/benny/39031159-%E5%BB%A3%E5%91%8A%E7%AD%96%E7%95%A5%E5%96%AE-%E9%9D%92%E7%AE%AD
  • https://www.facebook.com/l.php?u=http%3A%2F%2Fwww.magazine.org.tw%2Fevents%2F089%2Fstrategyplan_monrymg.doc&h=8AQGgzmUa

4.  希望會議結束後可以訂出明確時程，一步一步執行

  • 目前依照共筆等資料設計組發現，似乎除了設計組夥伴有較多接觸印刷經驗外其他組較少。
  • 在印刷上，請大家不要自行想像很恐怖啊～一天印刷更不可思議（模造類紙光放乾就不止了），還有是沒有一二三校和打樣......
  • 而通常印刷廠是可以一天完成印刷，但每家印刷廠都有本身的業務在，也是排時間性的，所以不可能你說要印她就馬上能幫你，大家都要乖該排隊的喔@@所以囉...如果問我到底幾天好，我不能跟各位保證...

5.  我們還是要爭取更長的製作天數!!

  • 文編+美編三天也太酷炫了= =實在是有一點太趕><"
  • 目前設計組這邊依照企劃組排成估算加進了印刷時間，再怎麼快，也是雙周刊，無法單周出刊，而設計組這邊如果要加上所謂的校稿、校對部分、那再快也至少要不含印刷，五個工作天??(關於這點組長我還要再和組員討論一下，TKS)

6. 跨組溝通

  • 當開會內容涉及他組業務，應邀請該組列席參與討論並提供咨詢，才能整合並有效率
  • 如討論到專欄內容應邀請文編一同開會，版面規劃找文&美編，通路管道找公關...etc
