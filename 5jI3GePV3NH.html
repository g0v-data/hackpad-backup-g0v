資料集領域:  都市計劃 

前言

據XXX表示，中央政府目前努力推動Open Data，但比較根本的問題市政府推出的資料不見得是民眾想要的，所以公開一些可有可無的資訊衝業績也不會得到什麼注意，因此應該把力氣花在市民關心的事上，於是目前高度注目的台南鐵路地下化爭議是個不錯的目標。

民眾對市政資料用途的想像

基本上可分為四個面向

  • 民眾為釐清社會爭議。
  • 民眾為了解政府施政，以供個人在財務，事業上作決策判斷。
  • 民眾為監督政府施政。
  • 民眾為開發第三方民間消費應用。 

族群需求分析

  • User Stories 指出以這個政策為例，不同族群的期望資料，以及用途

  • 想釐清台南鐵路地下爭議的民眾
    •
    • 注重第一手資料及真實性，
    • 應該要能取得所有重要相關電子公文，計劃文件以釐清決策脈絡。
    • 應該要能取得施工範圍的資料，知道影響範圍。
    • 應該要能取得土地徵收範圍的資料，知道會被徵收的土地分布圖。
    • 應該要能取得施工範圍內的地籍資料，以釐清是否有財團或人不當得利。
    • 應該要能取得鐵路地下化施工期間可能衝擊各領域的評估資料。
    • 應該要能取得各座談會，各公聽會，政府記者會的影音資料。
    • 應該要能取得政府澄清新聞稿的永久連結，發言單位，發言時間，佐證資料。
      • 佐證資料最好需要
        • 法律，法規，行政命令的永久連結。
        • 相關政府公文的永久連結。

    監督施政的民眾

    • 基本上同基本上需求同想釐清台南鐵路地下爭議的民眾。
    • 應該要能取得地下話工程施工預算及其細目，承標單位，施工狀態
  • 受施工影響的民眾
    •
    • 基本上需求同想釐清台南鐵路地下爭議的民眾。
  • 打算在台南買房子或開店的民眾
    •
    • 應該要能知道鐵路地下化的施工範圍
    • 應該要能知道鐵路地下化後的市景
    • 應該要能知道各段施工的工期，目前施工狀態，施工單位，施工單位通訊方式
    • 應該要能取得所有公聽會資料。
  • 想運用Data做第三方應用的民眾
    •
    • 基本需求同想釐清台南鐵路地下爭議的民眾，打算在台南買房子的民眾。
    • 對資料格式的需求為機器可讀。
