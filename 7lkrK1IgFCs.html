KKTIX 文案

RGBA × 柯文哲野生官網 unlimited 設計工作營

9月份的 RGBA 特別不一樣，「RGBA」與「零時前端教學松」、「JavaScript.tw」以及「柯文哲野生官網 unlimited」合作，在9月6日舉辦一整天的柯文哲野生官網 API 的應用設計與教學工作營！http://rgba.kktix.cc/events/unlimited

政治也能與技術結合？沒錯，政治應該能比以往更進步！難得，有市長候選人，能夠真正理解開放資料、開放 API 的價值，這次的活動，將以柯文哲野生官網 API 應用作為基礎，來讓更多人理解資訊技術與達成政治參與。替選舉或政治做出正面影響，需要大家一起 **鍵盤參政**

如果您對於網頁開發有興趣、也略懂基本 html，但對 Bootstrap 等各種前端資訊不瞭解或想做點東西、不知道該做什麼，我們誠摯的歡迎您加入我們活動的行列。活動開放柯文哲 API ，也歡迎混合各種技術，做出符合您想像的應用。

如果您是 App Developer 、 Maker 、裝置藝術家... 等，也非常歡迎加入交流，做出比網頁更厲害的應用！沒有任何限制！

這是一個開放的活動、在開放的環境進行，場地提供電源與無線網路，並供應早午餐。網站上有活動最基本的範例，不需要從零開始（例 http://tonyq.org/kp-wild/base/ ）。大家可以早上來想想自己想做什麼，如果有點成果（不論多、少）下午與大家一起分享。一起來體驗一個輕鬆又愉快的技術實作與簡單的政治參與。

活動中提供每個人 充足的電源插座、無線網路、簡單餐點。
