RGBA × 柯文哲野生官網 unlimited 設計工作營

RGBA 是每個月舉辦的網頁設計聚會，而 9 月份的 RGBA 與零時前端教學松、以及「柯文哲野生官網 unlimited」合作，在 9 月 6 日舉辦一整天的前端設計教學工作營！

當天會有人教您如何寫 HTML、CSS、Bootstrap 懶人包，還有邀請善用「柯文哲 API」寫出各種好玩野生官網、應用、開發工具的大師們，一起來交流聚會！

野生柯P官網，你來打造

以前有人說⋯⋯政府的網站很糟糕，設計得不夠好。所謂「官方」作的，不一定是最好的、也不一定是市民想要的， 甚至，對於什麼樣的網站才是「好」的，每一個人也都有不同的想像。 因此，我們將市長候選人的網站開放，我們不比稿、不找明星設計師， 我們想向市民徵求不同的想像、不同的創意，自己來製作心目中理想的「官方網站」、無限個不一樣的柯文哲官方網站。

這不是官網設計比賽，柯文哲會持續提供所有官方新聞、影音、照片的 API，讓設計師、工程師、自造者任意發揮。 做網站、做 App、甚至科技藝術的應用！ 想做 RWD 又方便盲人使用的網站？ 想做野生柯 P 的螢幕保護程式？ 想用四軸飛行器搭配 TTS 做新聞宣傳機？沒問題！ 所有的設計都是官方認可的官方網站，絕無高下之分！柯文哲的形象就由你來決定。

正面價值，開放透明，由你決定！

政治也能與技術結合？沒錯，政治應該能比以往更進步！難得，有市長候選人，能夠真正理解開放資料、開放 API 的價值，這次的活動，將以柯文哲野生官網 API 應用作為基礎，來讓更多人理解資訊技術與達成政治參與。替選舉或政治做出正面影響，需要大家一起**鍵盤參政**。

如果您對於網頁開發有興趣、也略懂基本 html，但對 Bootstrap 等各種前端資訊不瞭解或想做點東西、不知道該做什麼，我們誠摯的歡迎您加入我們活動的行列。我們活動開放柯文哲 API （請參考 http://unlimited.kptaipei.tw/docs ），也歡迎使用其他 API 或混合各種技術，做出符合您想像的應用。

如果您是 App Developer 、 Maker 、裝置藝術家... 等，也非常歡迎加入交流，做出比網頁更厲害的應用！沒有任何限制！

這是一個開放的活動、在開放的環境進行，場地提供電源與無線網路，並提供簡單餐點。網站上有活動最基本的範例，不需要從零開始（例 http://tonyq.org/kp-wild/base/ ）。大家可以早上來想想自己想做什麼，如果有點成果（不論多、少）下午與大家一起分享。一起來體驗一個輕鬆又愉快的技術實作與簡單的政治參與。
