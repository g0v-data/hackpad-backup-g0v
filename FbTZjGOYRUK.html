    • 公職人員財產申報
      • 有：url
      • 無 -> 未曾申報財產
    • 參選政治獻金專戶
      • 無 -> 未曾申請專戶
      • 有：列出歷次專戶彙總 table
        • 有專戶明細：末欄 link
        • 無專戶明細：末欄 調查兵團，我要認領 link
