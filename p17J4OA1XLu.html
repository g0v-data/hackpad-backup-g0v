UI Design (rough)

設計理念

  1. 在不影響UX的前提下增加UI的產品獨特性
  2. 由於涉及大量文字，使用能增加記憶點的排版風格
  3. 至少提供兩種配色，其中一種適合長時間文字閱讀
  4. 高效率、易使用、和操作概念一致的版面設計

使用元素及意象

concept : 

  • Use book / pages material felling
  • bookmarks - Media name and Comments 
  • charts
  • multiple chart-type display mode. (for diff-need-users)

Affordance 自釋性設計

色彩計畫

  • 主視覺色：紅
    • 代表熱情、積極、行動，能刺激使用者參與 (目前最常見媒體代表色)


  • 閱讀舒適
  • 易讀性高
