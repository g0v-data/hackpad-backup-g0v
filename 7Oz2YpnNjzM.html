#{急診及時等床資訊}

專案介紹 | Project Readme / Meta

因為台灣轉診制度不完全, 民眾習慣至較大醫院就醫, 導致急診雍塞

為方便民眾及基層轉診者瞭解並評估是否將會在急診等很久, 可以設計一網站, 讓使用者可以上傳最新帶床資訊, 供欲轉診者決定是否前往就診

來龍去脈（想解決什麼問題）

靈感來源: 

  1. 不時有人把照片放到FB, 但無整合
  2. 醫院並沒有義務把資訊公開到醫院網站
  3. 政府並沒有提供公開資訊讓民眾方便查詢

*

https://www.facebook.com/photo.php?fbid=372888479509233&set=a.118547204943363.20053.118446058286811&type=1&theater

主旨

建立一個網站或 App 可供上傳輸入以下資訊:

 #醫院 (可用GPS定位) #日期  #時間 (可抓手機系統資訊)

 #等候掛號人數 #等後看診人數 #目前急診人數  #等候住院人數 (手動輸入? 照片?)
