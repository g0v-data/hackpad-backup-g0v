irc 聊天紀錄設定方式

  1. 連上 freenode irc server
  2. 開一個符合活動或社群名稱的 irc channel
  3. 切換到 #g0v.tw channel
  4. mention au，請他替你的 irc channel 做 log
  5. 待 au 設定完成即可取得 logbot 網址
