軟體使用教學

Android - VLC for android Beta

  • 點選選單，選音訊

  • 點選右上角中間的天線符號

  • 輸入 http://i.summit.g0v.tw:8554/r0 (注意:圖片的port是錯的)

  • 完成



IOS - VLC

  • 要去點一個巨大的 menu 中的某個 open internet stream
  • 輸入http://i.summit.g0v.tw:8554/r0
  • 完成

IOS - Fstream

  • Favorites - > Edit - > Add new webradio
  • Name 隨便取
  • URL 輸入http://i.summit.g0v.tw:8554/r0
  • Encdoing 選UTF8
  • save 
  • Done
  • 回到Play 選 要的Name 按播放

MacOS - VLC

  • FILE -> open Network
  • 輸入http://i.summit.g0v.tw:8554/r0
  • 完成

WINDOWS - VLC

  • 媒體 -> 開啟網路串流
  • 輸入http://i.summit.g0v.tw:8554/r0
  • 完成
