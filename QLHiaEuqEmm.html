Code for Mexico City

（Laboratorio para la Ciudad）

  • Laboratorio para la Ciudad 為墨西哥市政府底下設立的市政實驗室。
  • http://labcd.mx/
  • 2013
  • 與 Code for America 和墨西哥市政府合作，在環境、健康、觀光、交通、經濟發展項目上合作，強調使用者中心，入選 6 位 programmer，提供發掘問題、工具使用等訓練課程。
  • 誕生的子專案舉例：
    • DataLab – 27 項整理後的開放資料，10 個 API。 http://datos.labplc.mx/
    • CHÉCATE AQUÍ ，HTTP://CHECATEAQUI.MX，協助民眾在地圖上就近搜尋免費的醫療資源
    • TRAXI ，HTTP://TRAXI.MX/，因墨西哥市內計程車參差不齊還有黑車，Traxi 提供搜尋車牌號碼得知其車輛檢測以及營業執照，甚至必要時可按下緊急求救按鈕。
  • 但是這些專案的連結很多都失效了QAQ
  • 這個市政 lab 下面有其他專案，例如 festival

KP Civic Innovation Fellowship, Code for Pakistan 

  • 2014 
  • 機制：與 Khyber Pakhtunkhwa 省政府和世界銀行合辦，每期招募 20 個公民科技社群人員（設計師、工程師），提供每月約 9000 台幣的補助、六個月的訓練、免費工作空間和政府協作。
  • 與政府合作成果：每期誕生5, 6 個專案
  • 誕生的子專案舉例：
    • 與地方災害管理處（ Provincial Disaster Management Authority (PDMA)）合作，建立網站，方便 NGO 、記者、民眾，線上提供資料以統計物資、贊助資金、各地受損狀況。

  • 讓交通警察可以回報交通事故並視覺化的 Android app

Code for Australia Fellowship Program

  • 成立年份：2016
  • 一年期，隨時可申請（on roll），送設計/工程人才與地方和中央政府合作，每月薪水 6250 澳幣（約 14 萬台幣）
  • 目前主要與維多莉亞省政府合作
  • 子專案舉例：
    • 與維多利亞省法律扶助處（Victoria Legal Aid）合作，提供半自動化的 bot 系統服務，民眾只需要輸入關鍵字詢問即會獲得所需資訊，大量節省電話客服與實體客服時間。
    • 與環境局合作（Department of Environment, Land, Water and Planning），改善老舊生物多樣性回報系統 Victorian Biodiversity Atlas，讓研究者能在 4 秒內迅速上傳資訊。
    • 與財政局合作，建立線上政府合約視覺化查詢（ Construction Contracts Register ）民眾可以檢視各項建設承包合約。

幾個有趣小觀察：

  • 這幾個 fellowship 都是在 city government level 合作
  • 許多是改善政府服務，這台灣比較少
  • 許多專案並不 high tech，跟 Code for America 一樣，有些系統只是改成 SMS 簡訊系統，就大幅縮短與政府接洽服務的流程。
  • 墨西哥市的雖然設在市政府底下，但反而專案有很多不必要跟政府合作相關，像是幫助民眾控制體重的 app。
  • 台灣的沒有人進入政府感覺更多是在進行基礎建設，觀念傳播，比較沒有明確的 app 成果展示
  • 網頁上都寫得很漂亮，但會想要知道具體的合作機制，這些外部人才是用什麼機制與政府合作。外包？標案？顧問？沒有正式職位但是會跟政府固定開會？
