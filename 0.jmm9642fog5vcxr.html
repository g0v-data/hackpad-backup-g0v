附錄：編輯試算表時，可以在儲存格內使用的語法

# 註解 (任何格子)

任一格子裡，開頭是 # 號代表這個格子是註解，標註解以後就可以亂寫，不會影響到資料，所以也不會影響到左邊的選單。

google spreadsheet id (A1)

A1 格子的內容如果不是註解，而且又 >= 40 字元，會被當作是 google spreadsheed id 解讀，

#A1: less than 40 characters, or..        #if you want to use google spreadsheet to save foldr data (usually when authorization is needed), fill in A1 with google spreadsheet id, and the same url will be redirected to gsheet.                

#url or item level. <: following links is level 0 (= top level)        #title        #option: hide/unsortable(foldr title), expand(subfoldr), blank(link)        #label: (color) content. color: gray/black/green/red/blue/orange/purple/teal/yellow/pink/deep-blue/deep-green/deep-purple/
